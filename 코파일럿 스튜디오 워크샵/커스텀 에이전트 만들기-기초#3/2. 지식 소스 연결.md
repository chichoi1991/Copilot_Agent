## Copilot Studio 지식 소스(Knowledge)란?

지식 소스는 에이전트가 사용자 질문에 답변할 때 참고하는 **내부 문서, 데이터 저장소**를 의미합니다.  
지식 소스를 연결하면 에이전트가 일반 언어 모델 지식 외에 **조직 내부 데이터**를 기반으로 답변할 수 있습니다.

| 지식 소스 유형 | 주요 사용 사례 |
|---------------|---------------|
| SharePoint | 사내 문서, 매뉴얼, 보고서 검색 |
| OneDrive | 개인 파일 검색 |
| 파일 업로드 (PDF, Word 등) | 특정 문서 고정 참조 |
| 공개 웹 URL | 외부 웹페이지 참조 |
| Dataverse 테이블 | 구조화된 데이터 질의 |

이번 실습에서는 **SharePoint** 와 **메일 커넥터** 를 연결합니다.

<br>

## 1. SharePoint 문서 지식 소스 연결

### 1-1. 참조 자료 추가 시작

에이전트 개요 화면에서 스크롤을 내려 **참조 자료** 섹션을 찾습니다.  
**+ 참조 자료 추가** 버튼을 클릭합니다.

<br>

### 1-2. SharePoint 선택

팝업 창에서 **SharePoint** 를 선택합니다.

<br>

### 1-3. SharePoint 경로 입력

SharePoint 사이트 주소 또는 특정 라이브러리 폴더 경로를 입력합니다.

정확한 경로를 확인하려면:
1. SharePoint 사이트 → 해당 라이브러리 폴더로 이동
2. 우측 상단 **세부 정보** 클릭
3. 사이드 패널 하단의 **경로** 복사

경로를 붙여넣고 **추가** 를 클릭합니다.

<br>

### 1-4. 지식 소스 설명 입력

추가된 지식 소스의 **설명** 란에 아래 내용을 입력합니다.  
설명이 구체적일수록 에이전트가 해당 지식 소스를 적절한 상황에서 활용합니다.

```
이 참조 자료는 사내 업무 문서, 제품 매뉴얼, 정책 자료가 포함된 SharePoint 라이브러리입니다.
사용자가 업무 관련 내용 또는 내부 규정을 질문할 때 이 자료를 우선 참고합니다.
```

<br>

### 1-5. 웹 검색 활성화 (선택)

참조 자료 섹션에서 **웹 검색** 토글을 **활성화** 합니다.  
웹 검색을 활성화하면 사내 문서에 없는 정보를 Bing 검색으로 보완합니다.

<br>

## 2. 메일 커넥터 연결

메일 커넥터는 에이전트가 사용자 메일함을 조회하거나 메일을 발송할 수 있도록 합니다.  
커넥터는 지식 소스가 아닌 **도구(Tools)** 로 등록됩니다.

### 2-1. 도구 추가

에이전트 개요 화면에서 **도구** 섹션으로 이동 후 **+ 도구 추가** 를 클릭합니다.

<br>

### 2-2. 커넥터 유형 선택

도구 추가 팝업에서 **커넥터** 탭을 선택합니다.  
검색창에 `Office 365 Outlook` 을 입력하고 해당 커넥터를 선택합니다.

<br>

### 2-3. 필요한 액션 선택

이번 실습에서는 아래 두 가지 액션을 추가합니다.

| 액션 | 용도 |
|------|------|
| **Send an email (V2)** | 메일 발송 |
| **Get emails (V3)** | 메일 목록 조회 |

각 액션을 선택하고 **연결** 버튼을 클릭하여 계정 인증을 완료합니다.

<br>

### 2-4. 도구 설명 보강

추가된 각 도구의 설명을 아래와 같이 수정합니다.  
설명이 구체적일수록 에이전트가 올바른 상황에서 도구를 호출합니다.

**Send an email** 설명:
```
사용자가 요청한 수신자에게 메일을 발송합니다. 수신자 이메일, 제목, 본문이 필요합니다.
```

**Get emails** 설명:
```
사용자의 받은 편지함에서 최근 메일 목록을 조회합니다. 특정 발신자 또는 키워드로 필터링할 수 있습니다.
```

<br>

## 3. 동작 확인

우측 테스트 패널에서 아래 질문을 입력하여 지식 소스와 메일 연결을 확인합니다.

**SharePoint 지식 소스 테스트:**
```
사내 문서에서 [문서 키워드]에 관한 내용을 찾아줘
```

**메일 도구 테스트:**
```
최근에 받은 메일 3개를 보여줘
```

에이전트가 SharePoint 문서를 인용하거나 메일 목록을 조회하면 2단계 완료입니다.
