# 6. Excel 데이터 필터링 + AI Prompt 연동 (심화)

> **이전 단계:** [5. AI Prompt 도구 추가](./5.%20AI%20Prompt%20도구%20추가.md) | **다음 단계:** [7. Teams 채널 배포](./7.%20Teams%20배포.md)

> ⚠️ **심화 실습입니다.** 기본 실습(1~5단계)을 모두 완료한 후 진행하세요.

---

## 이번 실습의 목표

SharePoint에 저장된 Excel 파일에서 **사용자 상황·수준에 맞는 데이터 행(Row)을 필터링**하고,  
해당 데이터를 **AI Prompt 도구에 전달**하여 맞춤형 답변을 생성하는 패턴을 구현합니다.

```
사용자 질문
    ↓
에이전트가 사용자 수준·상황 파악
    ↓
Excel 데이터 조회 (SharePoint 연결)
    ↓
조건에 맞는 Row 필터링
    ↓
필터링된 데이터 → AI Prompt 도구 전달
    ↓
맞춤형 HTML 응답 생성
```

<br>

---

## 1. Excel 데이터 준비

### 1-1. Excel 파일 구조 설계

실습용 Excel 파일을 SharePoint에 업로드합니다.  
파일은 아래와 같은 구조를 권장합니다.

| 열(Column) | 설명 | 예시 값 |
|-----------|------|---------|
| `ID` | 고유 식별자 | 1, 2, 3 ... |
| `Category` | 데이터 카테고리 | 신입사원, 중급자, 전문가 |
| `Level` | 난이도/수준 | 기초, 중급, 고급 |
| `Title` | 항목 제목 | "Excel 기초 사용법" |
| `Content` | 상세 내용 | 설명 텍스트 |
| `Tags` | 검색 태그 | "Excel, 업무도구, 기초" |

> 실제 실습에서는 강사가 제공한 Excel 파일을 사용합니다.

<br>

### 1-2. SharePoint에 Excel 업로드

1. SharePoint 사이트 → 문서 라이브러리 이동
2. **업로드** 버튼으로 Excel 파일 업로드
3. 파일 경로 메모 (지식 소스 등록 시 필요)

<br>

---

## 2. Excel 지식 소스 연결

### 2-1. 참조 자료 추가

에이전트 개요 → **참조 자료** → **+ 참조 자료 추가** → **SharePoint** 를 선택합니다.

업로드한 Excel 파일이 있는 SharePoint 경로를 입력합니다.

### 2-2. 지식 소스 설명 입력

```
이 참조 자료는 사용자 수준별 업무 가이드 데이터가 담긴 Excel 파일입니다.
Category 열로 대상자를 구분하고, Level 열로 난이도를 구분합니다.
사용자의 수준과 상황에 맞는 Row를 찾아 Content를 참고하여 답변합니다.
```

<br>

## 3. Excel 필터링 전용 AI Prompt 도구 추가

Excel 데이터를 받아 사용자 맞춤형 응답을 생성하는 별도 AI Prompt 도구를 추가합니다.

### 3-1. 새 AI Prompt 도구 추가

도구 → **+ 도구 추가** → **AI Prompt** 를 선택하고 아래와 같이 구성합니다.

**도구 이름:**
```
Personalized_Content_Generator
```

**도구 설명:**
```
사용자의 수준과 상황 정보를 받아, Excel에서 필터링된 데이터를 맞춤형 학습/업무 가이드 형태로 재구성합니다.
사용자 수준에 맞는 콘텐츠를 제공해야 할 때 이 도구를 사용합니다.
```

**입력 변수:**

| 변수명 | 유형 | 설명 |
|--------|------|------|
| `user_level` | 텍스트 | 사용자 수준 (신입사원 / 중급자 / 전문가) |
| `user_context` | 텍스트 | 사용자가 요청한 내용 또는 상황 설명 |
| `filtered_data` | 텍스트 | Excel에서 필터링된 Row 데이터 (JSON 또는 텍스트 형식) |

**프롬프트:**

```
당신은 맞춤형 업무 가이드를 제공하는 전문가입니다.

## 입력 정보
- 사용자 수준: {{user_level}}
- 사용자 요청/상황: {{user_context}}
- 관련 데이터: {{filtered_data}}

## 작성 규칙
1. 사용자 수준에 맞는 언어와 설명 깊이를 사용합니다.
   - 신입사원: 쉬운 용어, 단계별 설명, 예시 풍부하게
   - 중급자: 핵심 위주, 팁과 주의사항 포함
   - 전문가: 간결하게, 고급 활용법 중심

2. 제공된 데이터에서 사용자 상황과 가장 관련성 높은 내용을 선별합니다.

3. 아래 HTML 형식으로 출력합니다.

<div style="max-width:600px; font-family:Segoe UI, sans-serif; border-radius:6px; overflow:hidden; border:1px solid #E1DFDD;">
  <div style="background:#107C10; padding:12px 16px;">
    <span style="color:white; font-size:16px; font-weight:600;">
      {{user_level}} 맞춤 가이드
    </span>
  </div>
  <div style="padding:16px; background:#FFFFFF;">
    <!-- 사용자 맞춤 콘텐츠 -->
  </div>
  <div style="padding:8px 16px; background:#F3F2F1; font-size:12px; color:#605E5C;">
    수준: {{user_level}} | 관련 데이터 기반 생성
  </div>
</div>
```

## 4. 지침 업데이트

에이전트 지침에 아래 워크플로우를 추가합니다.

```
### 📊 맞춤형 콘텐츠 요청 시
사용자가 학습 자료, 업무 가이드, 수준별 안내를 요청하면:
1. 사용자의 수준을 파악합니다.
   - 사용자가 명시한 경우: 그 수준을 사용합니다.
   - 명시하지 않은 경우: 대화 맥락으로 추론하거나 사용자에게 질문합니다.
     예: "어떤 수준의 내용을 원하시나요? (신입사원 / 중급자 / 전문가)"
2. SharePoint Excel 지식 소스에서 해당 수준(Level 또는 Category)에 맞는
   데이터를 검색하여 관련 Row의 내용을 수집합니다.
3. 수집된 데이터와 사용자 맥락을 Personalized_Content_Generator 도구에 전달합니다.
4. 생성된 HTML 응답을 사용자에게 전달합니다.
```

## 5. 동작 확인

**수준 명시 테스트:**
```
신입사원인데, Excel 업무 활용법을 알려줘
```
→ 에이전트가 신입사원 수준 데이터를 필터링 → Personalized_Content_Generator 호출 → HTML 응답 반환

**수준 미명시 테스트:**
```
업무에서 자주 쓰는 도구 사용법 알려줘
```
→ 에이전트가 수준을 역질문 → 사용자 답변 후 → 맞춤형 콘텐츠 반환

**고급 활용 테스트:**
```
전문가 수준의 데이터 분석 방법론을 정리해줘
```
→ 전문가 수준 데이터 필터링 → 간결하고 고급 내용 위주로 HTML 응답

<br>

> 에이전트가 데이터를 필터링하는 조건(사용자 수준, 상황)을 지침에 명시하면,  
> Power Automate 없이도 동적 데이터 기반 맞춤형 응답이 가능합니다.
