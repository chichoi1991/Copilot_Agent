## 2. 참조자료 & 기능 추가하기

> **전제조건**: 에이전트 생성 및 지침 작성이 완료된 상태에서 진행합니다.

---

## 📚 Part 1: 참조자료 및 기능 추가하기

### 1. 참조자료(Knowledge) 추가

참조자료는 에이전트가 응답을 생성할 때 참조하는 데이터 소스입니다. 에이전트의 답변 품질과 정확성을 높이는 핵심 요소입니다.

#### 1.1 SharePoint 사이트 추가

**단계:**

1. 에이전트 편집 화면에서 **"참조 자료 추가"** 버튼을 클릭합니다 [1](https://hope.pe.kr/820)
2. 참조 자료 유형 선택 창에서 **"SharePoint"**를 선택합니다
3. SharePoint 사이트 URL을 입력합니다
   - 예시: `https://yourcompany.sharepoint.com/sites/YourSite`
4. **"추가"** 버튼을 클릭합니다
5. 사이트 인덱싱이 완료될 때까지 기다립니다

**팁:**
- 에이전트는 연결된 SharePoint 사이트의 문서, 리스트, 페이지를 자동으로 검색합니다
- 사용자는 본인이 접근 권한을 가진 콘텐츠만 조회할 수 있습니다 [2](https://learn.microsoft.com/ko-kr/microsoft-copilot-studio/knowledge-copilot-connectors)

---

#### 1.2 공개 웹사이트 추가

**단계:**

1. **"참조 자료 추가"** > **"공개 웹 사이트"** 선택 [1](https://hope.pe.kr/820)
2. 웹사이트 URL 입력
   - 예시: `https://learn.microsoft.com`
3. **"추가"** 버튼 클릭
4. 확인 체크박스 선택 후 **"추가"** 클릭
5. 크롤링 완료 대기

**주의사항:**
- 공개적으로 접근 가능한 웹사이트만 추가 가능합니다
- robots.txt 정책을 준수하는 사이트만 크롤링됩니다

---

#### 1.3 OneDrive 파일 추가

**단계:**

1. **"참조 자료 추가"** > **"OneDrive"** 선택
2. 특정 폴더 또는 파일 선택
3. **"추가"** 버튼 클릭

**지원 파일 형식:**
- 문서: `.docx`, `.pdf`, `.pptx`, `.xlsx`
- 텍스트: `.txt`, `.md`

---

#### 1.4 Copilot 커넥터 추가 (엔터프라이즈 데이터)

Copilot 커넥터를 통해 타사 엔터프라이즈 시스템의 데이터를 연결할 수 있습니다 [2](https://learn.microsoft.com/ko-kr/microsoft-copilot-studio/knowledge-copilot-connectors).

**단계:**

1. **"참조 자료 추가"** 클릭
2. 사용 가능한 Copilot 커넥터 목록에서 선택
   - 목록에 없는 경우 **"고급"** 선택
3. 원하는 커넥터 선택 후 **"에이전트에 추가"** 클릭

**사전 요구사항:**
- 테넌트 관리자가 해당 커넥터를 미리 구성해야 합니다
- Microsoft 365 Copilot 라이선스가 필요할 수 있습니다 [3](https://learn.microsoft.com/en-us/microsoft-365-copilot/extensibility/knowledge-sources)

---

### 2. 기능(Capabilities) 추가

#### 2.1 코드 인터프리터 (Code Interpreter) 활성화

코드 인터프리터를 통해 에이전트가 데이터 분석, 계산, 차트 생성 등을 수행할 수 있습니다.

**단계:**

1. 에이전트 설정 화면에서 **"기능"** 섹션으로 이동
2. **"코드 인터프리터"** 토글을 **ON**으로 설정
3. 변경 사항 저장

**활용 사례:**
- Excel 데이터 분석 및 시각화
- 복잡한 수학 계산 수행
- Python 코드 실행을 통한 데이터 처리

---

#### 2.2 이미지 생성기 (Image Generator) 활성화

에이전트가 사용자 요청에 따라 이미지를 생성할 수 있도록 합니다.

**단계:**

1. **"기능"** 섹션에서 **"이미지 생성기"** 선택
2. 토글을 **ON**으로 설정
3. 저장

**활용 사례:**
- 프레젠테이션용 이미지 생성
- 아이디어 시각화
- 마케팅 콘텐츠 초안 제작

---

#### 2.3 웹 검색 활성화

최신 정보를 검색할 수 있도록 웹 검색 기능을 활성화합니다.

**단계:**

1. **"기능"** > **"웹 검색"** 활성화
2. 필요시 **"범위 지정 웹 검색"**을 통해 특정 도메인으로 제한 가능 [3](https://learn.microsoft.com/en-us/microsoft-365-copilot/extensibility/knowledge-sources)

**주의:**
- 라이선스에 따라 일부 기능이 제한될 수 있습니다

---

### 3. 참조자료 및 기능 관리 팁

**참조자료 우선순위 설정:**
- 여러 참조자료를 추가한 경우, 에이전트는 가장 관련성 높은 소스를 자동으로 선택합니다
- 지침에서 특정 소스 우선 사용을 명시할 수 있습니다

**예시 지침:**
```
SharePoint의 "제품 매뉴얼" 사이트를 우선적으로 참조하여 답변하세요. <br> 
정보를 찾을 수 없는 경우에만 웹 검색을 사용하세요. <br>
```

**성능 최적화:**
- 참조자료는 최대 10개까지 추가 권장
- 너무 많은 소스는 응답 시간을 지연시킬 수 있습니다
- 정기적으로 사용하지 않는 참조자료를 제거하세요
